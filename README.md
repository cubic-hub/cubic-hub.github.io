# Cubic Hub Community Site

Cubic Hub is a community-driven website for **Cubic Castles** players. Right now the site focuses on two core features:

- **Crafting Page** - browse recipes and crafting information. 

- **Prices Page** - search up clothing prices (manually updated from Community Prices)
